# DAA-Final-Project

## La Empresa de Talentos Rusa

En TalentBridge Connect, operamos como un mercado de talento freelance de vanguardia, conectando a empresas con los profesionales independientes más cualificados para sus proyectos tecnológicos. Nuestra misión es simplificar la búsqueda de talento y asegurar que nuestros clientes obtengan los equipos perfectos para sus necesidades, de manera eficiente y rentable.

Nos enfrentamos a un desafío estratégico clave que impacta directamente nuestra capacidad para ofrecer el mejor servicio a nuestros clientes. Cuando un cliente nos presenta un nuevo proyecto, este viene con una serie de requisitos de habilidades muy específicos. Por ejemplo, un proyecto de desarrollo web podría necesitar un experto en "desarrollo frontend" con un nivel de dominio de 4 o superior, un "especialista en bases de datos" con un nivel 3 o superior, y un "diseñador UX/UI" con un nivel 5.

Nuestra plataforma cuenta con una vasta red de freelancers, cada uno con un perfil detallado que incluye su tarifa horaria y una evaluación de su nivel de dominio en diversas habilidades. Si un freelancer no ha sido evaluado en una habilidad particular, asumimos que no la posee para los fines del proyecto.

El problema que necesitamos resolver es el siguiente: ¿Cómo podemos seleccionar el equipo de freelancers más adecuado de nuestra red para cada proyecto de cliente, de modo que se cumplan todos los requisitos del proyecto al menor costo posible?

Esto implica tomar decisiones críticas:

- Para cada habilidad que el cliente solicita, debemos asegurarnos de que al menos un freelancer en el equipo seleccionado posea esa habilidad con el nivel de dominio mínimo o superior requerido. No basta con tener la habilidad; el nivel de experiencia es crucial.
- Dado que cada freelancer tiene una tarifa horaria, nuestro objetivo es que la suma total de las tarifas horarias de los freelancers seleccionados para el equipo sea la más baja posible. Queremos ofrecer a nuestros clientes la solución más competitiva sin comprometer la calidad.

Necesitamos una solución inteligente que nos permita analizar los requisitos de un proyecto y, a partir de nuestro pool de talento, identificar el conjunto óptimo de freelancers que, en conjunto, cubran todas las necesidades de habilidad con el nivel adecuado, minimizando al mismo tiempo el costo total para el cliente.
